Research projects

Jun 2021 - present
Food and Nutrition Science division, Department of Life Sciences, Chalmers University of Technology
TriplotGUI - An advanced visualization tool for molecular epidemiology (lead, ongoing)
This project aims to design a visualization tool (an R package) with shiny’s graphical user interface to explain the complex relationship between environmental exposures, human metabolic profiles, and disease outcomes. I designed the tool and wrote all the code for the UI and server and most of the code for the statistical models behind the interface. (https://gitlab.com/YingxiaoYan/triplotgui)

MUVR 2.0, a multivariate analysis tool for variable selection and prediction that minimizes overfitting across modeling strategies. (lead)
Combining machine learning and epidemiology modeling, this project aims to design a multivariate data analysis tool (an R package), MUVR 2.0, for unbiased variable selection while allowing confounding adjustment. Current MUVR 2.0 is in a branch of the original MUVR package. I wrote code for all the new functionalities in MUVR 2.0. ( https://gitlab.com/CarlBrunius/MUVR/-/tree/MUVR2)

StatTools, an R package of handy tools for random statistics(lead)
This package aims to develop handy tools for random statistics, which will aid data analysis and visualization in other research projects. My version of StatTools is originally forked from StatTools’ older version. I wrote all the new functions in the package since 2022 (https://gitlab.com/YingxiaoYan/StatTools)

OMICs signatures linking persistent organic pollutants to cardiovascular disease in the Swedish Mammography Cohort
This project aims to investigate the underlying mechanisms that link persistent organic pollutants; genetic, proteomics, and metabolomics profiles to cardiovascular diseases. I performed the data analysis (RStudio) and feature identification in this project.
